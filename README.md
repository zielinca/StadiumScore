StadiumScore App for Final Project of Prof John Gallaugher's SwiftUI Course at Boston College!
Contact me at: zielinca@bc.edu

Built using Swift and SwiftUI in Xcode for iOS; utilizes JSON/API data

StadiumScore is an MLB stadium rating app that lets users track the ballparks they’ve visited and rate each one across categories like atmosphere, food, and amenities. Users can upload photos, add their thoughts, view overall stadium rankings, and explore all MLB parks on an interactive map to see where they’ve been and where to go next.

StadiumScore utilizes an API/JSON file I created: https://zielinca.github.io/mlb-stadium-api/stadium.json
