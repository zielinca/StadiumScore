StadiumScore is an MLB stadium rating app where users track the ballparks they've visited and rate each one across categories like atmosphere, food, and amenities. Users can upload photos, leave notes, and see rankings of all stadiums. A map view lets users explore all MLB parks to see where they have and have not been.

StadiumScore utilizes an API/JSON file I created: https://zielinca.github.io/mlb-stadium-api/stadium.json
